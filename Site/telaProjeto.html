Como funciona o projeto ONDO?

Nosso projeto será implementado dentro da estufa de wasabi, ele estará coletando a temperatura a todo momento, e fará um trabalho com esses dados, para gerar as informações. Com essa informação o nosso cliente poderá tomar decisões acertivas para gerenciar a temperatura do ambiente. Utilizamos Sensores de Temperatura para gerar esses Dados e Graficos.Alem de gerar avisos e notificações, como se a temperatura está boa ou ruim. E temos uma Calculadora para simular lucro.
